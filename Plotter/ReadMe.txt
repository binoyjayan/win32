[		: To unsharpen the plot
]		: To sharpen the plot

RIGHT		: To shift the shape right 
LEFT  		: To shift the shape left 
UP    		: To shift the shape up 
DOWN  		: To shift the shape down 

HOME	 	: Zoom in
END    		: Zoom out

PAGE UP  	: stretch x 
PAGE DOWN 	: contrast x

INSERT		: To amplify the shape
DELETE 		: To de-amplify the shape

ESC		: To reset to default values

Left mouse btn: shift origin.

x: Rotate along x axis. 
y: Rotate along y axis.
r: Change the range of plot.

Speed up the operation using 'Ctrl' key.
